# OpenCV.js 集成开发指南

## 当前状态

OpenCV.js 已经安装并可以正常使用。我们创建了一个测试页面来验证基本功能。

## 测试方法

1. 访问 http://localhost:5173/opencv-test.html
2. 上传测试图片
3. 使用按钮测试不同的图像处理功能

## 下一步开发计划

### 1. 纸张检测功能实现

需要实现以下核心功能：
- 边缘检测
- 四边形拟合
- 透视校正
- 纸张尺寸标定

### 2. 集成到主应用

将 OpenCV.js 功能集成到校准页面中：
- 自动检测纸张四角
- 手动调整功能
- 透视校正处理
- 尺寸标定计算

## OpenCV.js 使用注意事项

1. OpenCV.js 是一个大型库，加载时间较长
2. 需要在 Web Worker 中运行复杂的计算以避免阻塞 UI
3. 注意内存管理，及时释放 Mat 对象
4. 某些功能可能需要 wasm 支持
