# 测试工作流程

## 基本功能测试

1. **访问应用**
   - 打开浏览器访问 http://localhost:5173

2. **上传图片**
   - 点击文件上传按钮
   - 选择 `testpic.jpg` 文件
   - 验证图片是否正确显示在纸张检测页面

3. **导航测试**
   - 点击步骤条中的各个步骤按钮
   - 验证每个页面是否正确显示
   - 验证图片在各个页面中是否正确显示

4. **功能按钮测试**
   - 在纸张检测页面点击"自动检测纸张"按钮
   - 在纸张检测页面点击"手动调整"按钮
   - 在AI分割页面点击"一键 AI 分割"按钮
   - 在AI分割页面点击图片区域

## 预期结果

1. 应用正常启动并在端口 5173 上运行
2. 上传页面显示正确的上传指引
3. 上传后自动跳转到纸张检测页面
4. 图片在所有相关页面中正确显示
5. 所有按钮点击后显示相应的提示信息
6. 步骤导航功能正常工作
